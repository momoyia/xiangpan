相盼 App UI 视觉设计标准规范 v1.0

优雅美学与科学运动的完美平衡 • 专为年轻一代打造的数字健身美学空间

1. 品牌视觉设计哲学 (Design Philosophy)

我们致力于打破传统健身应用枯燥的数据罗列和冰冷、强迫式的机械打卡。其核心设计哲学包含三个维度：

轻盈通透的沉浸感：大面积的留白设计，配合柔和的动态投影与细腻的磨砂高斯模糊效果，消除界面的厚重与压迫感。

温情科技美学：使用充满朝气与梦幻感的优雅紫到樱花粉的自然渐变，激发运动多巴胺，让每一次流汗都像星光般闪耀，充满仪式感。

极简主义排版：遵循经典杂志的排版艺术，使用大胆的大字号标题、非对称网格瀑布流和模块化轻量级卡片，让信息层级自然、顺畅地聚焦。

2. 品牌色彩系统 (Color System)

为保证界面的高级感，色彩系统必须被严格限制在以下色域内，以纯净白为底色，确保主色彩的高饱和度与活力。

2.1 品牌色 (Brand Colors)

渐变起始色（优雅紫）：#8A31EB

设计释义：象征星空的深邃、智慧、科技、静谧与专注。

渐变截止色（樱花粉）：#FA84D2

设计释义：象征朝气、温暖、活力、星光陪伴与新生。

渐变应用规范：角度统一采用 135°，主要用于核心行为按钮、顶栏高亮图标、数据进度条以及重点强调的背景卡片。

2.2 基础与中性色 (Neutral Colors)

画布底色（纯白）：#FFFFFF

应用：全局页面背景。

浅灰背景层（次背景）：#F8FAFC (Tailwind slate-50)

应用：次要表单底板、卡片默认底色、分割模块。

主文本色（深夜蓝）：#0F172A (Tailwind slate-900)

应用：大标题、重点文字。

次要文本色（石墨灰）：#475569 (Tailwind slate-600)

应用：副标题、段落正文。

辅助文本色（浅烟灰）：#94A3B8 (Tailwind slate-400)

应用：时间标签、辅助提示、未激活状态 Tab。

4. 网格与空间网格系统 (Grid & Spacing)

星叙采用严密的 8px 网格系统，所有的间距、组件高度、内边距必须是 8 的倍数（或特殊微小组件采用 4 的倍数）。

[8px 栅格与 safe area 触控边界图示]
|<- 20px ->| [====卡片内容====] |<- 20px ->|


全局侧边安全边距 (Safe Area Margin)：统一为 20 px，确保手机屏幕边缘留出完美的视觉空白。

卡片内边距 (Padding)：统一为 16 px（或紧凑排版下使用 12 px）。

卡片纵向间距 (Gap)：

瀑布流卡片间距：16 px

社区 Feed 流卡片间距：24 px

最小点击热区尺寸：所有交互对象（如按钮、点赞星星、导航切换、设置按钮）的点击物理区域不得低于 44 &times; 44 px，以确保在快速运动（如跑步、行走）中单手操作的准确性。

5. 组件与质感系统 (Components & Depth)

我们摒弃粗暴的黑色硬描边，采用模拟物理自然界的光影规律来建立视觉层级。

5.1 模块与卡片圆角 (Border Radius)

特大圆角 (24px)：用于首页焦点组件、发现页专项锻炼大卡片。

标准圆角 (16px)：用于普通瀑布流卡片、Feed 流展示图、我的饮食打卡卡片。

小型圆角 (12px / 8px)：用于头像框、表单输入框、功能标签、快速筛选徽章。

5.2 微弱投影规范 (Shadows)

轻悬浮态 (Soft-Shadow)：

box-shadow: 0px 10px 30px -10px rgba(138, 49, 235, 0.12);


应用：主页面底部导航栏、顶部状态模块。

无压力卡片投影 (Card-Shadow)：

box-shadow: 0px 15px 35px -5px rgba(0, 0, 0, 0.05);


应用：瀑布流卡片、个人数据模块、绿色配餐模块。

5.3 磨砂玻璃效果 (Blur)

在顶部导航栏与底部 Tab 栏上强制应用：

backdrop-filter: blur(20px);
background-color: rgba(255, 255, 255, 0.75);


设计效果：在滑动时，卡片背景会在导航栏下若隐若现，形成通透、轻盈的呼吸美感。

6. 微交互与阻尼动画规范 (Micro-interactions)

流畅且符合物理规律的动效，能够提升用户对健康管理App的安全感与掌控感。

Tab 页面转场动效：

参数：Transition-property: opacity, transform; Duration: 300ms; Curve: ease-in-out;

交互：轻点 Tab 键，旧界面微微缩小（scale: 0.98）并向后淡出，新界面从底部向上弹入（Y轴偏移 10px 到 0px）。

核心按钮触压态 (Active Feedbacks)：

当按下“立即开始训练”或“保存更改”时，按钮具有微小形变弹动：

transform: scale(0.96);
transition: transform 150ms cubic-bezier(0.25, 0.8, 0.25, 1);


情感化点赞/收藏交互：

点击心形/五角星瞬间，图标会经历一次先放大至 1.3倍 再回弹至 1.0倍 的反弹式呼吸动效。

7. 健康与绿色 A11y 绿色通道 (Healthy Design Guideline)

作为专为年轻用户设计的健康共享空间，星叙坚守健康积极的视觉调性：

绿色配餐配色：选用生机勃勃的薄荷绿（#10B981）辅助展示健康减脂膳食，杜绝压抑的减肥色调。

数据正向激励：所有图标均采用圆角化设计，避免锐利角度产生的数据焦虑感；统计条采用平滑端点（Round Cap），让目标达成过程显得柔和亲切。

安全屏障：不包含任何刺激性的闪烁动画，暗光环境下保持白色背景在 80% 以上高对比度的清晰可读性。
